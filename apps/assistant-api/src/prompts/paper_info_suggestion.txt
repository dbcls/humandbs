# タスク説明
論文情報をもとに、以下の内容を判定・抽出してください。
* 研究の研究内容・貢献の要約（日本語で100文字以内）。特に、対象患者や対象とした遺伝子・疾患などについて詳細に言及してください。
* 研究に関連するICD-10コードのリスト。論文が特定の疾病に対して言及していない場合は、ICD-10コードのリストを空にしてください。
* 研究に関連する解析手法のリスト。各要素は手法名の文字列。例: ['手法1', '手法2']
* 論文がヒト由来のデータ（患者情報、ヒト検体、ヒトゲノム・遺伝子情報、ヒト由来細胞など）を扱っているかを true/false で判定してください。
* ヒトのデータを扱っている場合は、その判断根拠となるタイトルまたは概要の該当箇所を evidence_excerpt にそのまま抜粋してください。
* ヒトのデータを扱っていない、もしくは根拠を抜粋できない場合は evidence_excerpt を空文字にしてください。
* ヒトデータを扱っているかどうかにかかわらず、なぜそう判断したかの理由を human_data_reason に日本語で簡潔に記述してください。
